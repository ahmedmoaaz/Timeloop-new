### Step 1: Create Some Events (If You Haven't Already)
1. Go to **Events** tab

3. Add a few events with:
   - Title (e.g., "Work on project")

## 🐛 Troubleshooting

### If AI Insights Still Doesn't Work:

1. **Check if you have events**:
   - Go to Events tab
   - Make sure you have at least 1 event logged today

2. **Check API key quota**:
   - Visit: https://platform.openai.com/account/usage
   - Verify you have credits/quota available

3. **Check billing**:
   - Visit: https://platform.openai.com/account/billing
   - Ensure payment method is added

4. **Look at the error message**:
   - The app will show specific error messages
   - Common errors:
     - "Quota exceeded" → Add billing/credits
     - "API key not configured" → Check .env file
     - "No data" → Add some events first
