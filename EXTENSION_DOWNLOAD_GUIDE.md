## 🚀 Quick Installation Guide

### Step 1: Download the Extension

**Go to this URL in your browser:**
```
https://productivity-ai-35.preview.emergentagent.com/download-extension
```

OR

**Direct download link:**
```
https://productivity-ai-35.preview.emergentagent.com/chrome-extension.zip
```

### Step 2: Extract the ZIP File

1. Find the downloaded `chrome-extension.zip` in your Downloads folder
2. Right-click on it
3. Select "Extract All..." or "Extract Here"
4. You'll get a folder called `chrome-extension`

### Step 3: Install in Chrome

1. Open Chrome browser
2. Type in address bar: `chrome://extensions/`
3. Turn ON "Developer mode" (toggle in top-right)
4. Click "Load unpacked"
5. Select the **chrome-extension folder** (the extracted folder)
6. Click "Select Folder"

### Step 4: Pin the Extension

1. Click the puzzle piece 🧩 in Chrome toolbar
2. Find "TimeLoop Tracker"
3. Click the pin icon 📌
4. Now you'll see ⏰ in your toolbar!

### Step 5: Configure

1. Click the ⏰ icon
2. Enter your User ID
3. Click "Save Configuration"

### Step 6: Start Tracking!

1. Browse to YouTube or any website
2. Stay for 10+ seconds
3. Switch to another tab
4. Click ⏰ icon
5. Check History tab!
